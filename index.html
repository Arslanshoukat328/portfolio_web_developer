About Me

Who Am I?

I've spent the last five years helping businesses and individuals turn raw data into something useful. As a freelancer, I've built web scraping systems that extracted millions of records from sites like Amazon and LoopNet—which meant I had to truly understand how each website worked, whether the data was static, dynamically loaded, or hidden behind APIs. I've created desktop applications that let clients watch their data come in live, and built automation pipelines that run reliably on servers.

Lately, I've been focused on web development, working with Django and React to build full-stack applications from the ground up. I'm the kind of developer who likes to understand how things work, then make them better—like when I took an open-source password manager, added encrypted fields, and deployed it securely for remote access. I'm driven by curiosity, clarity, and code that just works.
